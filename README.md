# Python校园用车管理系统简介

## 核心功能
- 学生信息管理
- 校车信息管理
- 教师信息管理
- 司机信息管理
- 校车记录管理
- 学生乘坐管理
- 教师乘坐管理
- 个人中心

## 适用场景
- 校园内部用车管理
- 教师与学生出行记录管理
- 校车调度与安排

## 技术特点
- 使用 Django 框架进行后端开发
- 使用 MySQL 数据库进行数据存储
- 使用 Vue.js 进行前端开发
